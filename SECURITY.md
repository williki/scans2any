# Security Policy

## Supported Versions

Currently, only the latest release of `scans2any` is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

Instead, please report them via email to the project maintainers or by using GitHub's "Report a vulnerability" feature under the Security tab. Please provide a descriptive summary of the issue, the steps to reproduce, and the potential impact.

We will endeavor to respond to your report within 48 hours and will keep you informed of our progress towards a fix and formal announcement.
